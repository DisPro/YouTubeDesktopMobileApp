# YouTube Desktop Mobile App

## Description
This is a mobile application that brings the YouTube Desktop experience to mobile devices. It leverages the YouTube Music API to provide users with access to their favorite music and videos in a desktop-like environment on their mobile devices. The app allows users to control playback (play, pause, stop, previous, next) directly from their phones.

## Features
- Browse and search for music videos
- Control playback (play, pause, stop, previous, next) directly from your phone
- Seamless integration with YouTube Music platform
- Personalized recommendations
- Dark mode support for comfortable viewing at night
